# shipment-analysis-dashboard
Interactive Shipment Analysis Dashboard using Excel, SQL and Power BI
Shipment Analysis Project Report

Objective:
To analyze shipment data and generate actionable insights to improve delivery performance, reduce delays, and support data-driven decision-making using analytics and visualization techniques.

Dataset:
Shipment dataset (CSV) containing Shipment ID, Product Category, Customer & Region details, Shipment Mode, Delivery Status & Time.

Tools & Technologies:
Python (Jupyter Notebook)
 Pandas, NumPy
Matplotlib, Seaborn, Scikit-learn
Power BI / Excel.

Key Work Performed:
- Cleaned and preprocessed raw shipment data
- Performed Exploratory Data Analysis (EDA)
- Built visualizations to identify trends and patterns
- Applied ML models (Random Forest, Linear Regression) to predict delays
- Designed interactive dashboard with KPIs and filters

Dashboard Highlights:
- KPIs: Total Shipments, On-Time %, Avg Delivery Time
- Region-wise and mode-wise shipment analysis
- Delivery status distribution
- Interactive filters for dynamic insights

Key Insights:
- Certain regions experience higher delivery delays
- Air mode is fastest but cost-intensive
- Peak periods lead to increased shipment delays
- Product category impacts delivery time

Business Impact:
- Improved delivery efficiency
- Better logistics planning
- Reduced delays and operational costs
